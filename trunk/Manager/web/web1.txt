function getCoords(el) {
	var box = el.getBoundingClientRect(),
		doc = el.ownerDocument,
		body = doc.body,
		html = doc.documentElement,
		clientTop = html.clientTop || body.clientTop || 0,
		clientLeft = html.clientLeft || body.clientLeft || 0,
		top = box.top + (self.pageYOffset || html.scrollTop || body.scrollTop) - clientTop,
		left = box.left + (self.pageXOffset || html.scrollLeft || body.scrollLeft) - clientLeft;
	return { 'top': top, 'left': left };
};

<p>事故船只停靠在苏州太湖水上搜救基地内接受调查 新华社发</p>

<p>东方网4月7日消息：据《新闻晚报》报道，“我们少一个人！ ”“没有！ ”“我们有一个人可能掉下去了！ ”“没有，没有，肯定没有！ ”“求求你们开一圈，去找一下好不好？ ”“没有！ ”……任凭学生如何声嘶力竭地苦苦哀求，可肇事驾驶员始终<a href="http://discover.news.163.com/special/helpinghand/" title="冷漠" target="_blank">冷漠</a>回绝。昨天傍晚，江苏卫视播出了太湖事发现场的一段视频，随后视频在网上被疯传，视频中显示，有学生苦求游艇驾驶员救人时，驾驶员否认有人失踪，不肯去搜救。</p>

<p>记者上午从苏州吴中区宣传部证实，目前肇事驾驶员已被正式刑事拘留。</p>

<p><strong>驾驶员连连否认有人落水</strong></p>

<p>据江苏卫视报道，当时，岸边的游客正悠闲地享受假日，突然，一艘面目全非的快艇出现在大家的视野里。船上的人一个个头破血流，还清醒的几个人大呼救命。岸边的游客也一下子没反应过来，船上和岸上的血迹触目惊心。有人开始救援，也有人开始逃避责任。面对不可收拾的局面，快艇驾驶员却好像不顾人命，只顾着隐瞒事实。岸上的人苦苦哀求驾驶员，但他却一直坚持没有人落水。</p>

<p>记者在这段视频中清楚地看到如下对话，学生一开始说，“我们少一个人！ ”船老大强势回应，“没有！ ”“我们少一个人！ ”“没有，没有，肯定没有！ ”“你们开船去找一下好吗？有一个人估计掉下去了！ ”“有一个人不见了！ ”……“没有，没有，绝对没有！ ”“求求你再开一圈去找一下，好不好？ ”……学生从一开始的声嘶力竭到最后明显带着哭腔的苦苦哀求，听来着实让人揪心、痛心，可船老大依然无动于衷。</p>

<p>记者粗略数了一下，整段视频中，船老大一共说了8个没有，坚决否认有人落水的事实。连江苏卫视的主持人在节目中都气愤到无法控制情绪，破口大骂：“找一下，你会死啊！很抱歉，逃避和冷漠让我无法控制！ ”</p>

<p>随后，这段视频也在网上疯传引起热议，学生声嘶力竭的痛苦哀求却被肇事者冷漠回绝，面对此情此景，不少网友气到爆。尽管在视频中无法最真切看清楚船老大的面部表情，但从他的声音中听到了除了冷漠，还是冷漠。这种冷漠是对生命的漠视，是对责任的推卸。</p>

<p><strong>肇事者涉嫌危险驾驶</strong></p>

<p>昨天下午，苏州市领导和有关部门负责人向家属通报了事故调查的初步结果。记者了解到，初步结果显示，在两船间距比较小的情况下，强行穿越属于<a href="http://baoxian.163.com" title="违章">违章</a>行为，乘客未穿救生衣驾驶员开船涉嫌危险驾驶。目前，肇事者已被刑事拘留。</p>
